Hero
============
Terminal based file manager for linux

Features
--------
* UTF-8 Support
* Up to 9 different Tabs

Features
--------
* UTF-8 Support
* Up to 9 Tabs

Getting Started
--------
'python hero.py'

Usage
--------
Navigation:  

Use the Arrow Keys or 'h' 'j' 'k' 'l'  

Switch Tabs:  

Use the Number Keys from 1-9  

Perform actions  


| Key           | Action        |
| ------------- |:-------------:|
| q             | Quit          |
| c             | Copy          |
| p             | Paste         |
| d             | Delete        |
| r             | Reload        |
| t             | Rename        |
| k             | Create(File/Folder)|
