Hero
============
Terminal based file manager for linux

Features
--------
* UTF-8 Support
* Up to 9 Tabs

Getting Started
--------
`python hero.py` 

Usage
--------
Navigation:  

&nbsp;&nbsp;Use the Arrow Keys or `h` `j` `k` `l` 

Switch Tabs:  

&nbsp;&nbsp;Use the Number Keys from 1-9  

Perform actions:  


| Key           | Action        |
| ------------- |:-------------:|
| q             | Quit          |
| c             | Copy          |
| p             | Paste         |
| d             | Delete        |
| r             | Reload        |
| t             | Rename        |
| k             | Create(File/Folder)|
